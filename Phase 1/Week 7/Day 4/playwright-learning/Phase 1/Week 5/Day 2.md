## Types of Software Testing

### 1. Unit Testing

In this testing we are checking each unit is working correctly or not this type of checking was done by developer

### 2. Integration Testing

In this testing we are checking that different - different unit are working correctly in a combine way and modules import/export are working correctly. this testing was done by tester.

Ex : Test that login function correctly connect to DB.

### 3. System Testing

In this type of testing complete application or software will be tested againt a documnet or provided guidelines. It was tested by testers and it's check that software meet business requirements. or you can say test entire application as a whole.

Ex : Test Complete Checkout flow end to end.

### 4. Uers Acceptance Testing (UAT) or Acceptance Testing

In this type of testing we check that software will fulfill end customer needs and customer check or business people check that it meets all requirements that a business claming.
