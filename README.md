# js-automation-toolkit

A modular, scalable end-to-end testing and automation framework. This repository contains the core utilities, type-safe architecture, and Playwright execution suite built to ensure high software reliability across UI and API layers.

## 🏗️ Tech Stack

- **Core:** JavaScript (ES6+), TypeScript, Node.js
- **Framework:** Playwright
- **CI/CD:** GitHub Actions / Docker
- **API:** RESTful (Postman / Playwright Request Context)

## 📂 Architecture & Implementation Sprints

This framework is being engineered in modular phases (tracked incrementally by sprints/days):

- **`/00-JavaScript-Fundamentals`**
  - Core JS utilities, data structures (Arrays/Objects), and custom algorithmic logic. Includes modules for test-data generation, math operations, and DOM manipulation basics.
- **`/01-TypeScript-Advanced`**
  - Type-safe interfaces, custom types, and Object-Oriented Programming (OOP) implementations (Page Object Model foundations) for framework stability.
- **`/02-Playwright-API`**
  - The primary execution engine. Contains end-to-end UI automation, network interception, and hybrid REST API integration tests.
- **`/03-Testing-Theory`**
  - Documentation and implementation of test case design techniques (Boundary Value Analysis, Equivalence Partitioning) and Agile QA methodologies.
- **`/04-DevOps-Git`**
  - Infrastructure as Code (IaC). CI/CD pipeline workflows, Dockerization scripts, and continuous testing configurations.
- **`/05-System-Design`**
  - High-level scalable test architecture, reporting infrastructure (Allure/HTML), and parallel execution strategies.
